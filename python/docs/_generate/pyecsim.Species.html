pyecsim.Species¶

class pyecsim.Species(name: str, conc: float, diff: float) → pyecsim.Species¶

Create chemical species with name, concentration and diffusion coefficient

Parameters

  • name (str) – name of the species (can be anything)

  • conc (float) – initial concentration [mol/m^3 or mM]

  • diff (float) – diffusion coefficient [m^2/s]

Returns

newly generated species

Return type

pyecsim.Species

Attributes

normConc¶

Returns the concentration normalized to all species’ concentrations

normDiffCoeff¶

Returns the diffusion coefficient normalized to all species’ coefficients

normEquilConc¶

Returns the normalized concentration after equilibration
